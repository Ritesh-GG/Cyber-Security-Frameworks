# Cyber Security Frameworks 
## Overview Research and comparison of Australian cyber security frameworks relevant to public sector and enterprise organisations in Victoria. 
## Frameworks Covered 
- VPDSF (Victorian Protective Data Security Framework)
- ASD Essential Eight Maturity Model
- ISO 27001 Information Security Management
## Documents 
- Framework-Comparison.md - Side by side comparison of all 3 frameworks including control mapping, data classification comparison and governance roles 
## Key Findings 
The VPDSF is mandatory for Victorian public sector and requires annual attestation. 
The Essential Eight provides a maturity model approach to cyber attack mitigation. 
ISO 27001 provides the broadest international framework with third party certification. 
All three are complementary and can be implemented together.
